# LegalDefendAI

A comprehensive legal assistance platform built with Streamlit that helps users navigate Indian legal processes, manage documents, and access legal resources.

## Features

- Document Management System with OCR capabilities
- Legal Code Search and Bookmarking
- Case Search and Precedent Analysis
- Jurisdiction Assistant
- Rights Predictor
- Amendment Tracker
- Hearing Scheduler

## Project Structure

```
LegalDefendAI/
├── app.py                      # Main Streamlit app
├── requirements.txt            # List of Python dependencies
├── README.md                   # Project documentation
├── .streamlit/                 # Streamlit config folder
│   └── config.toml
├── pages/                      # Streamlit pages
├── data/                       # Data files
├── assets/                     # Static assets
└── utils/                      # Helper functions and modules
```

## Setup

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Run the application:
```bash
streamlit run app.py
```
